# Machine Learning and Pattern Recognition 🤖

This repository contains labs and the project developed during the course of **Machine Learning and Pattern Recognition** at **Politecnico di Torino**. The objective is the implementation from scratch of algorithms for data analysis, dimensionality reduction and classification.

## 📂 Repository structure

The repository is organized in two macro folders:
* `Labs`: Implementation of the different topics seen in the course
* `Project`: Project developed during the course
